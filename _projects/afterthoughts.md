This site started as a bare Jekyll repo with a README and has since become something I actually want to use.

## Goals

- A single place to document everything I'm doing and learning
- Low friction to write and publish
- Doesn't look like every other developer blog

## Stack

Built with Jekyll and plain HTML/CSS.

## What I learned

Static site generators are genuinely underrated for personal sites. The cognitive overhead is low, the output is fast, and you can version-control everything including your content.
